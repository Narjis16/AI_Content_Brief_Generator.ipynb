AI Content Brief Generator

A web app that generates SEO-optimized content briefs for any topic using **Groq's Llama 3** and **Streamlit**.

 Live Demo
"https://proximity-armhole-decimal.ngrok-free.dev"

 Features

Enter any topic (e.g., "sustainable gardening", "AI in healthcare") and get:

- 🎯 **Target Audience** – 3 personas who would read this content
- 🔑 **SEO Keywords** – 5 keywords to optimize for search engines
- 📚 **Blog Outline** – H1 title + 3 section headings
- 🎭 **Tone of Voice** – Suggested writing style

 Technologies Used

| Technology | Purpose |
|------------|---------|
| Groq (Llama 3 70B) | Fast LLM API for content generation |
| Streamlit | Interactive web interface |
| Python | Backend logic |
| ngrok | Temporary public URL (for Colab deployment) |
